## Sobre

Projeto **Pokédex**, projeto feito para simular uma pokédex dos jogos da franquia Pokémon utilizando a [PokéAPI](https://pokeapi.co/)

---

## Tecnologias utilizadas

O projeto foi feito utilizando das seguintes tecnologias:

- React JS
- Typescript
- Styled-Components
- Axios

---

## Funcionalidades

- Listagem de pokémons da api separados por região

- Visualização dos tipos de cada pokémon

- Visualização das cores alternativas de cada pokémon
